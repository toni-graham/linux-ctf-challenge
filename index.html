Welcome to The Encrypted Labyrinth CTF Challenge

You are a new administrator for a server at a secret research lab. The lab has recently completed an important research project, but the research director left behind a cryptic encrypted message and specific instructions on how to find the research results. Your job is to uncover the hidden flag.

Step 1: Access the Server

The first step is to log into the server. Use the following credentials:

Username: researcher

Password: lab@2024

After logging in, you'll be in the home directory of the user. You need to explore the files there to start your journey.

Step 2: Investigate the Hidden File

After logging in, use the following command to list hidden files:

ls -la

You should see a hidden file named .message.txt.

Inside the .message.txt file, you will find a cryptic message:

The code lies in the puzzle of access control. Look closely at the permissions.

Step 3: Adjust Permissions

To access the content of .message.txt, you need to adjust its permissions. Use the following command:

chmod 644 .message.txt

Once you change the permissions, you can read the file using:

cat .message.txt

The file will tell you that the flag is encrypted and a cipher key is hidden somewhere in the environment variables.

Step 4: Find the Cipher Key

To find the cipher key, run:

env

You should see a hidden environment variable:

CIPHER_KEY=42

Step 5: Decrypt the Message

The message is encrypted using a Caesar cipher. The key is 42.

The encrypted message you will find in .encrypted_message.txt is:

Wkh jvssr xp lqzhe qeb czkpv. Wkh nbyz xlmw ql wkh wpkbzx.

Step 6: Access the Flag

Go to the /desk directory and use the following command to access it:

sudo chmod 755 /desk

Once you access the desk directory, look for the hidden file .flag.txt.

Inside .flag.txt, you will find the final flag:
